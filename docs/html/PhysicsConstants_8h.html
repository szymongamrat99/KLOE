class  PhysicsConstants
 Manages and provides access to experiment-specific configuration. This class combines compile-time constants with runtime-loaded parameters from a JSON file. All runtime parameters are loaded into strongly-typed member variables during construction for easy and type-safe access. This class is NOT a Singleton; it's designed for Dependency Injection.
