Reconstruction of neutral vertex using the triangle method.

Parameters
TrcSumFinalpointer to variable to get the TrcSum after the method application $$^{4}_{i=1} (T_{cl,i} - {d_{,i}}{c} - {d_{K}}{c{K}})$$
vtxSigmaFinalpointer variable to get the average error of the neutral vertex reconstruction (over clusters' energies) $${^{4}_{i=1}E_{cl,i}{(X_{neu,i} - X_{neu,avg})^2 + (Y_{neu,i} - Y_{neu,avg})^2 + (Z_{neu,i} - Z_{neu,avg})^2}}{^{4}_{i=1}E_{cl,i}}$$
Clu5Vecmatrix of clusters' variables:
  • first index: choice of cluster
  • second index:
    • 0-3: ($X_{cl} [cm], Y_{cl} [cm], Z_{cl} [cm], T_{cl} [ns]$)
    • 4: $E_{cl} [MeV]$
ippointer to the array of interaction point's spatial coordinates: ($X_{IP}, Y_{IP}, Z_{IP}$)
Phi4Mompointer to the array of $$ meson's 4-momentum: ($p_{x}, p_{y}, p_{z}, {s}$)
Kne4Mompointer to the array of kaon's 4-momentum: ($p_{x}, p_{y}, p_{z}, E_{K}$)
Kne4Vecpointer to the array of kaon's 4-position: ($x_{neu,avg}, y_{neu,avg}, z_{neu,avg}, t_{K}$)
trcpointer to the array of $$(T_{cl,i} - {d_{,i}}{c} - {d_{K}}{c{K}})$$ for each cluster
Returns
The function returns:
  • 0: no errors
  • ErrorHandling::ErrorCodes::DELTA_LT_ZERO
  • ErrorHandling::ErrorCodes::DENOM_EQ_ZERO
